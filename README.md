# Week 2 GenAI Workflow

## Business Workflow
This project converts unstructured meeting or operational notes into clear, structured action items.

## User
The user is a healthcare project manager or operations lead who needs to track follow ups from meetings.

## Input
The system receives unstructured notes from meetings, discussions, or workflow updates.

## Output
The system produces a structured list of action items that includes:
- the task
- the owner if available
- the priority level (High, Medium, Low)

## Value
This task is valuable because important follow ups are often lost in long or messy notes. Automating this process improves accountability, saves time, and helps teams stay aligned.
